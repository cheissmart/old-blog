<center><h1>a + b</h1></center>

#### Discription:
```markdown
input two integers(a and b)
output the sum of the two integers(a + b)
```

### a_plus_b.cpp
```markdown
#include <iostream>

using namespace std;

int main()
{
  int a, b;
  cin >> a >> b;
  cout << a + b << endl;
}
```

### a_plus_b.c
```markdown
#include <stdc++.h>

int main()
{
  int a, b;
  scanf("%d%d", &a, &b);
  printf("%d\n", a + b);
}
```
